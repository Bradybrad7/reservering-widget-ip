Embed Code voor Externe Websites

üìã Complete Embed Code

Kopieer onderstaande code en plak deze in uw website waar u de reserveringswidget wilt tonen:

<!-- Inspiration Point Reserveringswidget -->
<div id="inspiration-point-widget-container"></div>

<!-- React Dependencies (CDN) -->
<script src="https://unpkg.com/react@18/umd/react.production.min.js" crossorigin></script>
<script src="https://unpkg.com/react-dom@18/umd/react-dom.production.min.js" crossorigin></script>

<!-- Widget Styles -->
<link rel="stylesheet" href="https://unpkg.com/inspiration-point-reservation-widget@1.0.0/dist/style.css">

<!-- Widget Script -->
<script src="https://unpkg.com/inspiration-point-reservation-widget@1.0.0/dist/reservation-widget.umd.js"></script>

<script>
  // Callback functie wanneer reservering voltooid is
  function handleReservationComplete(reservationData) {
    console.log('Reservering voltooid:', reservationData);
    
    // Voeg hier uw eigen code toe om de data te verwerken
    // Bijvoorbeeld: stuur naar uw backend, toon een bedankbericht, etc.
    
    // Optioneel: Toon een bevestigingsbericht
    // alert('Bedankt voor uw reservering!');
    
    // Optioneel: Redirect naar een bedankpagina
    // window.location.href = '/bedankt';
  }

  // Optionele configuratie
  const customWidgetConfig = {
    maxCapacity: 200,              // Maximum aantal personen per reservering
    minPersons: 15,                 // Minimum aantal personen
    termsUrl: '/algemene-voorwaarden',  // URL naar algemene voorwaarden
    privacyUrl: '/privacy',         // URL naar privacy statement
    // ... meer opties mogelijk
  };

  // Initialiseer de widget
  window.addEventListener('DOMContentLoaded', function() {
    const container = document.getElementById('inspiration-point-widget-container');
    
    if (container && window.ReservationWidget) {
      ReactDOM.render(
        React.createElement(window.ReservationWidget.ReservationWidget, {
          onReservationComplete: handleReservationComplete,
          config: customWidgetConfig  // Optioneel: verwijder deze regel als u geen custom config gebruikt
        }),
        container
      );
    } else {
      console.error('Kon de widget container of de widget library niet vinden.');
    }
  });
</script>

};

üîß Troubleshooting

‚ö†Ô∏è Veelvoorkomende problemen:

Widget wordt niet geladen

  • Controleer of alle script tags correct zijn
  • Open de browser console (F12) en check op fouten
  • Zorg dat React scripts voor de widget script worden geladen
  • Controleer of de container div het juiste ID heeft

Styling ziet er vreemd uit

  • Zorg dat de style.css wordt geladen
  • Check of uw eigen CSS conflicteert met de widget styles
  • Gebruik browser DevTools om te inspecteren

Console errors

  • ReservationWidget is not defined - Widget script is niet geladen of in verkeerde volgorde
  • React is not defined - React CDN scripts ontbreken of verkeerde volgorde
  • Container not found - Controleer het ID van de container div
